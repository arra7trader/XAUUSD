# XAUUSD Extreme Expert

Project ini sedang diposisikan ulang menjadi platform analisa XAUUSD yang fokus pada:

- naked chart,
- market structure,
- liquidity behavior,
- session behavior,
- dan driver khusus emas.

## Dokumen Inti

- [docs/XAUUSD_DEEP_REFERENCE.md](./docs/XAUUSD_DEEP_REFERENCE.md)
- [docs/XAUUSD_NAKED_CHART_FRAMEWORK.md](./docs/XAUUSD_NAKED_CHART_FRAMEWORK.md)

Dua dokumen ini adalah fondasi knowledge base untuk membangun assistant analisa XAUUSD dan website yang lebih sesuai dengan arah proyek.

## Development

Jalankan server lokal:

```bash
npm run dev
```

Build production:

```bash
npm run build
```

Lint:

```bash
npm run lint
```

## Arah Produk

Target produk bukan lagi kursus trading umum, tetapi mesin bedah XAUUSD yang bisa membantu membaca:

- bias harga,
- market state,
- key levels,
- liquidity sweep,
- acceptance / rejection,
- dan skenario bullish / bearish / neutral.
